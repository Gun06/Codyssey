# 🏴‍☠️ 우주 해적 웹서버

Python의 `http.server` 모듈을 사용하여 구현한 HTTP 웹서버입니다. 우주 해적단을 소개하는 아름다운 웹페이지를 제공합니다.

## 🚀 빠른 시작

```bash
# 서버 실행
python3 http_server.py

# 웹 브라우저에서 접속
http://localhost:8080
```

## ✨ 주요 기능

### 기본 기능

- 🌐 **8080 포트 HTTP 서비스** - 표준 웹 포트에서 서비스 제공
- ✅ **200 OK 응답** - 정상적인 HTTP 상태 코드 전송
- 📄 **우주 해적 웹페이지** - 아름다운 HTML 페이지 제공
- 📊 **접속 로그** - 실시간 접속 정보 모니터링

### 보너스 기능

- 🌍 **IP 위치 정보** - 접속자의 지역 정보 자동 조회
- 🔒 **사설 IP 구분** - 로컬/외부 네트워크 구분 처리
- 🎨 **반응형 디자인** - 모든 디바이스에서 최적화된 화면
